## LoopLab Studio: test_fusion_comp

`test_fusion_comp.setting` is a simplified Fusion composition for testing LoopLab Studio procedural animation tools. 

1. Download the [test_fusion_comp.setting](./test_fusion_comp.setting) file.
1. Start a new project and switch to the Fusion page.
1. Drag `test_fusion_comp.setting` to the canvas.
1. Add the LoopLab tool to the canvas. 
1. Link one of the `Test_` nodes to the tool's `Curve` and test.
